Scorecraft API Documentation

Signup a Teacher

  • URL: /api/teacher/signup
  • Method: POST
  • Description: Creates a new teacher account.
  • Request Body:
    • name (string, required): The name of the teacher.
    • profileurl (string): Public Url of Profile
    • clientid (string,required): Google Auth clientid
    • email (string, required): The email address of the teacher.
    • institution (string, required): The institution the teacher is associated with.
    • verified (Boolean, required): true if auth successful
  • Responses:
    • 201 Created: Teacher account created successfully.
    • 400 Bad Request: Invalid request body or missing required fields.
    • 500 Internal Server Error: Server error.

Signin a Teacher

  • URL: /api/teacher/signin
  • Method: POST
  • Description: Logs in a teacher
  • Request Body:
    • email (string, required): The email address of the teacher when account created.
  • Responses:
    • 200 OK: Login successful. Returns a JWT token.
      • success (number): 1 if login is successful.
      • message (string): "Login Successful".
      • token (string): JWT token for authentication.
    • 401 Unauthorized: Invalid credentials or account not verified.
      • success (number): 0 if login is unsuccessful.
      • message (string): Error message detailing the reason for failure.
    • 500 Internal Server Error: Server error.
